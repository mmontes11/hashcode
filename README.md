# Hashcode

Google hashcode code challenge by `Los Almendros`.

### Installation

Install [node](https://nodejs.org/) and [nvm](https://github.com/nvm-sh/nvm). Afterwards run:

```
$ npm i
$ nvm install
$ nvm use
```

### Start

```
$ npm start
```

### Debug

```
$ npm run start:inspect
```

### Profile

```
$ npm run start:profile
```
